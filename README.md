# React Todo App ✅

A simple and clean Todo List application built with React.  
It includes features to **add**, **edit**, and **delete** todo items.

## 🚀 Features

- Add new todos
- Edit existing todos
- Delete todos
- Realtime updates with `useState`

## 🛠️ Tech Stack

- React
- JavaScript
